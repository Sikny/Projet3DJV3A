# Projet3DJV3A

Contrôles :

Clic gauche pour selectionner une unité (les unités alliées de couleur bleue) 

Une fois sélectionnée, clic gauche pour amener l'unité a un point de la carte.

Maintenir clic droit en bougeant la souris pour déplacer la caméra 

Molette de la souris pour zoomer/dézommer
